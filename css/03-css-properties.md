# 🖼 Propiedades CSS 

Podemos dividir las propiedades CSS en dos tipos:
- Apariencia 
- Organización 

---

## 🎨 Apariencia
Las propiedades más utilizadas son: 
- `color`
- `background-color`
- `background-image`
- `font-size`
- `border`
- `width`
- `height`
- `z-index `

## 📐 Organización
- `display`
- `position`
- `margin`
- `padding`
- `box-sizing`
